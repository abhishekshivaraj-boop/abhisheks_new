# E-Commerce Order Management System

Java backend application built using JDBC and MySQL to manage customers, products, and orders.

## Tech Stack
Java
JDBC
MySQL
OOP
DAO Pattern

## Features
- Product management
- Customer management
- Order creation
- Database integration
- CRUD operations

## Setup

1. Clone repository
2. Import project into IntelliJ / Eclipse
3. Run database.sql in MySQL
4. Update DB credentials in DBConnection.java
5. Run MainApp.java
